About This Demo

This interactive demo showcases a Retrieval‑Augmented Generation (RAG) pipeline alongside a generative AI agent. It allows you to compare answers from a vector‑search‑based RAG system and an LLM‑driven agent. The UI features a sleek glass‑morphism design, dark‑mode toggle, and micro‑animations for a premium feel.

Built with:

  • Flask + FastAPI (uvicorn) for the server
  • Weaviate as the vector store
  • Groq LLM for fast inference
  • Vanilla HTML/CSS/JS with a modern design system
